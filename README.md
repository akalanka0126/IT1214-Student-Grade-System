# IT1214-Student-Grade-System

**Project Purpose**
This project is a simple Java console-based application developed for academic purposes.
It helps a private educational institute manage student records digitally instead of using manual paper records or spreadsheets.

The system demonstrates the use of Object-Oriented Programming (OOP) concepts in Java.

**Features**

The system allows teachers to:
    Add new student records (ID, Name, Marks)
    Display all students
    Search for a student using Student ID
    Calculate the average marks of all students
    Exit the application
